# Espen's Skills

Well, not my _actual_ skills, but like… [Agent Skills](https://agentskills.io).

## Installation

```bash
pnpx skills add rexxars/skills --skill='*'
```

or to install all of them globally:

```bash
pnpx skills add rexxars/skills --skill='*' -g
```

Learn more about the CLI usage at [skills](https://github.com/vercel-labs/skills).

## Skills

- [rexxars](skills/rexxars): Espen's preferences and best practices for library projects (typescript, vitest, code style etc.)
- [vitest](skills/vitest): Use fewer tokens, don't re-run for different output, use `stubEnv` and other vitest helpers when helpful
